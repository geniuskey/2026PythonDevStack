# 머신러닝 & AI 라이브러리

## 카테고리

### 1. 딥러닝 프레임워크

| 라이브러리 | 등급 | 한 줄 설명 |
|-----------|------|-----------|
| **pytorch** | ⭐ | 현대 딥러닝 표준 프레임워크 |
| **tensorflow** | ⚠️ | 구글의 딥러닝 프레임워크 (PyTorch가 더 인기) |
| **jax** | 🧩 | 고성능 자동 미분 (연구용) |

### 2. LLM & AI 애플리케이션

| 라이브러리 | 등급 | 한 줄 설명 |
|-----------|------|-----------|
| **langchain** | ⭐ | LLM 애플리케이션 프레임워크 |
| **llamaindex** | ⭐ | LLM 데이터 프레임워크 (RAG 특화) |
| **openai** | ⭐ | OpenAI API 공식 클라이언트 |
| **anthropic** | ⭐ | Anthropic (Claude) API 클라이언트 |

### 3. 벡터 데이터베이스

| 라이브러리 | 등급 | 한 줄 설명 |
|-----------|------|-----------|
| **chromadb** | ⭐ | 로컬 벡터 DB (개발용) |
| **pinecone-client** | ⭐ | 클라우드 벡터 DB |
| **faiss** | ⭐ | Facebook의 벡터 검색 라이브러리 |

### 4. 전통 머신러닝

| 라이브러리 | 등급 | 한 줄 설명 |
|-----------|------|-----------|
| **scikit-learn** | ⭐ | 전통 ML 알고리즘 (분류, 회귀, 클러스터링) |
| **xgboost** | ⭐ | Gradient boosting (Kaggle 최애) |
| **lightgbm** | ⭐ | 빠른 gradient boosting |

### 5. NLP

| 라이브러리 | 등급 | 한 줄 설명 |
|-----------|------|-----------|
| **transformers** | ⭐ | Hugging Face 트랜스포머 모델 |
| **spacy** | ⭐ | 프로덕션 NLP 라이브러리 |
| **nltk** | ⚠️ | 교육용 NLP (spacy가 더 빠름) |

---

## 선택 가이드

### 딥러닝을 배운다면?
→ **pytorch** (커뮤니티 최대, 직관적)

### LLM 앱을 만든다면?
→ **langchain** (RAG, 에이전트) + **chromadb** (벡터 DB)

### 전통 ML (분류, 회귀)?
→ **scikit-learn** (쉽고 강력)

### NLP 프로덕션?
→ **spacy** (빠르고 정확) 또는 **transformers** (최신 모델)
