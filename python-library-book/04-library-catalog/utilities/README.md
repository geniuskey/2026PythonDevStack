# 유틸리티 라이브러리

## 주요 라이브러리

| 라이브러리 | 등급 | 분야 | 설명 |
|-----------|------|------|------|
| **pydantic** | Yes | 데이터 검증 | 타입 안전 데이터 검증 |
| **structlog** | Yes | 로깅 | 구조화 로깅 |
| **python-dateutil** | Yes | 날짜/시간 | 날짜 파싱 및 계산 |
| **arrow** | Yes | 날짜/시간 | 더 나은 날짜/시간 API |
| **python-dotenv** | Yes | 설정 | .env 파일 로드 |
| **rich** | Yes | 터미널 | 터미널 포매팅 (색상, 테이블 등) |
| **tqdm** | Yes | 프로그레스 바 | 진행률 표시 |
