# 데이터 과학 라이브러리

## 카테고리

### 1. 데이터프레임

| 라이브러리 | 등급 | 한 줄 설명 |
|-----------|------|-----------|
| **polars** | Yes | 고성능 데이터프레임 (pandas보다 10배 빠름) |
| **pandas** | Yes | 사실상 표준 데이터 분석 라이브러리 |
| **dask** | - 니치: | 병렬/분산 데이터프레임 (대용량 데이터) |

### 2. 데이터베이스

| 라이브러리 | 등급 | 한 줄 설명 |
|-----------|------|-----------|
| **duckdb** | Yes | 인메모리 분석 DB (SQL + DataFrame 통합) |
| **sqlalchemy** | Yes | ORM 및 SQL 툴킷 |

### 3. 시각화

| 라이브러리 | 등급 | 한 줄 설명 |
|-----------|------|-----------|
| **plotly** | Yes | 인터랙티브 시각화 |
| **matplotlib** | Yes | 기본 시각화 라이브러리 |
| **seaborn** | Yes | matplotlib 기반 통계 시각화 |

### 4. 수치 계산

| 라이브러리 | 등급 | 한 줄 설명 |
|-----------|------|-----------|
| **numpy** | Yes | 배열 및 수치 계산 기본 라이브러리 |
| **scipy** | Yes | 과학 계산 (통계, 최적화, 신호 처리) |

---

## 선택 가이드

### 데이터 분석 처음이라면?
→ **pandas** (생태계 최대, 튜토리얼 많음)

### 성능이 중요하다면?
→ **polars** (10배 빠름, 멀티코어 활용)

### SQL을 쓰고 싶다면?
→ **duckdb** (DataFrame에 직접 SQL 쿼리)

### 시각화?
→ **plotly** (인터랙티브) 또는 **matplotlib** (정적, 출판용)
