# 라이브러리 카탈로그

파이썬 생태계의 라이브러리들을 분야별로 정리한 레퍼런스 섹션입니다.

---

## 📚 카테고리

### [웹 개발](web-development/README.md)
프레임워크, HTTP 클라이언트, 템플릿 엔진, WSGI/ASGI 서버

**주요 라이브러리**: FastAPI, Django, httpx, uvicorn

### [데이터 과학](data-science/README.md)
데이터 분석, 시각화, 수치 계산

**주요 라이브러리**: polars, pandas, matplotlib, numpy

### [머신러닝 & AI](machine-learning/README.md)
딥러닝, NLP, 컴퓨터 비전, LLM 도구

**주요 라이브러리**: pytorch, langchain, transformers, scikit-learn

### [시스템 & 자동화](system-automation/README.md)
CLI, 스케줄링, 배포, 모니터링

**주요 라이브러리**: click, schedule, fabric, prometheus-client

### [문서 처리](document-processing/README.md)
PDF, Excel, Word, 텍스트 파싱

**주요 라이브러리**: pypdf, openpyxl, python-docx, beautifulsoup4

### [멀티미디어](multimedia/README.md)
이미지, 오디오, 비디오 처리

**주요 라이브러리**: pillow, opencv-python, ffmpeg-python

### [네트워킹](networking/README.md)
소켓, 프로토콜, 웹 스크래핑

**주요 라이브러리**: httpx, websockets, scrapy

### [보안 & 암호화](security/README.md)
인증, 해싱, 토큰 관리

**주요 라이브러리**: cryptography, pyjwt, python-jose

### [유틸리티](utilities/README.md)
날짜/시간, 로깅, 설정 관리, 유효성 검사

**주요 라이브러리**: pydantic, structlog, python-dateutil
